Introduction
------------
Activity Calendar Extension allows you to view all CiviCRM activities in a Calendar by month,day,week. The setting page allows us to select which activities should be shown on Calendar with the color we want for particular activity type. 

Installation
-------------
1. Move the downloaded extension to your extensions folder.
2. Goto civicrm/admin/extensions&reset=1  -- install the extension
 
If you are new to CiviCRM Extension you can get help about extension from

http://wiki.civicrm.org/confluence/display/CRMDOC42/Extensions

Library
---------------
Extension uses the fullcalendar library.
Add the fullcalendar library https://github.com/fullcalendar/fullcalendar/releases/tag/v1.6.7

Usage
---------------
1. Click Administrator->Customize Data and Screens->Activity Calendar Settings menu (civicrm/activityCalendarsettings) -- change the setting if required.
2. Click Contacts->Show Activities menu (civicrm/showactivities) -- to view Activity Calendar with Activities as per settings.

Note
-----
The Joomla and WordPress plugins have been removed as they are not needed by the organization for which the Activity Calendar was created.
